# Django Todo List App

A modern, user-friendly task management application built with Django. Manage your daily tasks efficiently with features for creating, updating, tracking, and completing tasks.

## Features

- **User Authentication** - Secure registration and login system
- **Task Management** - Create, update, and delete tasks
- **Task Tracking** - Dashboard to track daily, weekly, and monthly progress
- **Reminders** - Set date and time reminders for tasks
- **Responsive Design** - Works on desktop and mobile devices

## Tech Stack

- **Backend**: Python, Django 5.2
- **Frontend**: HTML, CSS, JavaScript
- **Database**: SQLite
- **Deployment**: Vercel

## Getting Started

### Prerequisites

- Python 3.11+
- Django 5.2

### Installation

1. Clone the repository:
```bash
git clone https://github.com/Shreyascode40/Todo-list-django.git
```

2. Navigate to the project directory:
```bash
cd Todo-list-django
```

3. Install dependencies:
```bash
pip install -r requirements.txt
```

4. Run migrations:
```bash
python manage.py migrate
```

5. Start the development server:
```bash
python manage.py runserver
```

6. Open your browser and go to `http://127.0.0.1:8000/`
